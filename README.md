# Covid-19 deep learning classification model
This PyTorch model classifies chest X-rays with the following possible classes: *covid*, *normal* or *viral_pneumonia*. Dataset was downloaded from Kaggle. It is avaliable uder 
this [link](https://www.kaggle.com/datasets/pranavraikokte/covid19-image-dataset). The images were classified by convalutional neural network. CNN contained 4 blocks containing convolutional, pooling and ReLU layers. 
It was trained in 30 epochs and reached 93% accuracy.
